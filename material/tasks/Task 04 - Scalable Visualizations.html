# Lab - Scalable Applications These tasks provide an opportunity for you to try out and experiment with the D3 features (e.g., customizing and modifying basic examples to specific problems and data sets). # References * [Objects](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object) * [Object Factories](https://www.javascripttutorial.net/javascript-factory-functions/) * [Classes](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Classes) # Encapsulating Visualizations Take the interactive visualization examples from the previous week and update them to take advantage of **object factories** (or classes) to encapsulate the creation/functionality, so you can create instances of the visualizations. For example, put 5 or 6 of the same graph on screen at the same time but each displaying different data (also send messages to the different graphs for different control aspects, such as, colors, sizing, position on screen and so on).
